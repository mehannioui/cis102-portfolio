My Projects

Smart Garden System

For my IoT course project, I developed a Smart Garden System that uses sensors to monitor soil moisture, light levels, and temperature. The system automatically waters plants when needed and adjusts artificial lighting to optimize plant growth.

Technologies used: Arduino, C++, IoT sensors, MQTT protocol

This project taught me valuable lessons in hardware-software integration and real-time data processing. It was particularly rewarding to see the system successfully maintain a small herb garden in our university's computer science lounge.

Natural Language Processing Chatbot

As part of my AI course, I created a chatbot that uses natural language processing to answer questions about our university's computer science program. The chatbot can handle queries about course requirements, faculty information, and departmental events.

Technologies used: Python, NLTK, TensorFlow, Flask

This project deepened my understanding of NLP techniques and machine learning models. It also improved my skills in web development as I created a user-friendly interface for interacting with the chatbot.

Data Visualization Dashboard

For a hackathon project, I led a team in creating a data visualization dashboard that displays COVID-19 statistics. The dashboard pulls data from various public health APIs and presents it in an interactive, easy-to-understand format.

Technologies used: JavaScript, D3.js, React, Node.js

This project was particularly challenging due to the time constraints of the hackathon and the need to work with real-time data. It significantly improved my skills in front-end development and data manipulation. Our team was proud to win second place in the hackathon with this project.
